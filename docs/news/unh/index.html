I was delighted to visit UNH and longtime colleague Laura Dietz to talk about formal language support for experimentation as part of the Robotics Seminar Series. Empiricism and evaluation methodology is important across the board in science, and it was fun to share the ways in which we can better support reproducibile data collection in the context of experimentation. It also doesn't hurt to have a secret ally to evangelize the power of formal languages via Ben Gamari.
